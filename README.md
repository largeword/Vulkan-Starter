# Vulkan-Starter
This project is used to get hands on Vulkan and configure CMakeList.

It was ripped from [Building a Triangle with Vulkan and C++ in Fedora 37](https://jordanielder.com/posts/building-a-triangle-with-vulkan-and-cmake-in-fedora/), original tutorial can be found there or in the local [tutorial](./tutorial) folder.

## Usage
Clone the repo and open it with Clion, then run.
